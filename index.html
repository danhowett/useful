Incident Command Console

The single-screen "war room" dashboard the IC opens during an active incident. Tracks elapsed time, severity, current phase, role assignments, decisions, actions, regulatory and external notifications, comms, and evidence — all in one place. Auto-builds a unified timeline from every event. Aligned with NIST SP 800-61r2 phases, with cross-references to NCSC, ISO 27035, and SANS PICERL. Floating Quick-Log button (or press Q) for one-click capture without leaving the screen. Opt-in browser auto-save; PIR-ready Markdown / JSON / PDF export.
